## Objective

Remove Instagram and X/Twitter social links from every page. No real REACH Equity Lab profiles exist for these platforms, so the placeholder hrefs must be deleted entirely. LinkedIn is the only real profile and must be kept. Two surfaces are affected:

1. **Footer `.socials` div** — present on all 7 HTML files
2. **Connect page social grid** — the `.connect-social-grid` in `contact.html` contains three `.connect-social-link` cards (LinkedIn, Instagram, X); remove the Instagram and X cards

Current footer pattern (all 7 pages):
```html
<div class="socials">
  <a href="https://www.linkedin.com" target="_blank" rel="noreferrer">LinkedIn</a>
  <a href="https://www.instagram.com" target="_blank" rel="noreferrer">Instagram</a>
  <a href="https://x.com" target="_blank" rel="noreferrer">X / Twitter</a>
</div>
```

Target footer pattern (all 7 pages):
```html
<div class="socials">
  <a href="https://www.linkedin.com" target="_blank" rel="noreferrer">LinkedIn</a>
</div>
```

Current Connect page social grid (contact.html lines 56–78):
```html
<div class="connect-social-grid">
  <a href="https://www.linkedin.com" target="_blank" rel="noreferrer" class="connect-social-link">
    <span class="connect-social-icon">in</span>
    <div>
      <strong>LinkedIn</strong>
      <span>Professional updates & publications</span>
    </div>
  </a>
  <a href="https://www.instagram.com" target="_blank" rel="noreferrer" class="connect-social-link">
    <span class="connect-social-icon">ig</span>
    <div>
      <strong>Instagram</strong>
      <span>Behind-the-scenes & events</span>
    </div>
  </a>
  <a href="https://x.com" target="_blank" rel="noreferrer" class="connect-social-link">
    <span class="connect-social-icon">𝕏</span>
    <div>
      <strong>X / Twitter</strong>
      <span>Research highlights & commentary</span>
    </div>
  </a>
</div>
```

Target Connect page social grid:
```html
<div class="connect-social-grid">
  <a href="https://www.linkedin.com" target="_blank" rel="noreferrer" class="connect-social-link">
    <span class="connect-social-icon">in</span>
    <div>
      <strong>LinkedIn</strong>
```
