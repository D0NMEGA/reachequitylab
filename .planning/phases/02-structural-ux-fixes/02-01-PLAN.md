## Objective

Verify that both contact pages serve distinct purposes, are correctly linked in the nav, and that the "Contact Us" CTA button is visually distinct from regular nav links. The context decision confirms both pages should be kept as-is: `contact.html` is the Connect hub (info + social grid + email) and `contact-us.html` is the form-only page. No content changes are needed — this plan audits the nav markup and confirms correctness, then documents any gap found.

Current nav on all pages:
```html
<a href="contact.html" data-transition>Connect</a>
<a href="contact-us.html" data-transition class="nav-cta">Contact Us</a>
```

The `nav-cta` class exists and is already applied to the Contact Us link. The audit confirms this is already correct. No structural changes are required for NAV-01 or NAV-02 — the plan closes both requirements by verification.
